SabayGo is a modern web app for commuters to share rides and split fares, built with React + TypeScript, TailwindCSS, shadcn/ui, and Supabase. Save money, reduce commuting stress, and connect with other riders going the same route — all in real-time!

Tech Stack
Frontend: React + TypeScript + Vite
Styling: TailwindCSS + shadcn/ui
Backend: Supabase (PostgreSQL, Auth, Realtime, Storage)
Realtime & Messaging: Supabase Realtime
Optional: Geolocation and Map API (Google Maps or Leaflet)

Features
Create a ride post (pickup, destination, time, available seats)
Browse available rides and join/request a ride
Real-time updates of rides and ride requests
User-to-user messaging
Optional current location detection and map-based ride selection
